Bab 1 — Apa itu HTML?

HTML adalah singkatan dari HyperText Markup Language. HTML adalah bahasa yang digunakan untuk membuat halaman web. Semua website yang kamu lihat di internet (Google, YouTube, Instagram) semuanya dibuat menggunakan HTML!

Bayangkan HTML seperti kerangka tulang dari sebuah website. Dia menentukan bagian-bagian halaman seperti judul, paragraf, gambar, dan tombol.

Tahukah kamu? HTML dibuat oleh Tim Berners-Lee pada tahun 1991. Sudah lebih dari 30 tahun dipakai di seluruh dunia!

HTML menggunakan tag untuk memberi tahu browser (seperti Chrome atau Firefox) cara menampilkan isi halaman. Tag ditulis menggunakan tanda kurung sudut < >.

<!-- Ini adalah contoh tag HTML sederhana --> <h1>Halo, Dunia!</h1> <p>Selamat datang di website pertamaku.</p>

Bab 2 — Struktur Dasar HTML

Setiap halaman HTML punya struktur dasar yang harus kamu ikuti. Seperti rumah yang punya fondasi, dinding, dan atap, HTML juga punya bagian-bagian penting:

<!DOCTYPE html> <html> <head> <title>Website Pertamaku</title> </head> <body> <h1>Halo!</h1> <p>Ini halaman web pertamaku.</p> </body> </html>

Penjelasan setiap bagian:

  • <!DOCTYPE html> — Memberitahu browser bahwa ini dokumen HTML5
  • <html> — Pembungkus seluruh halaman
  • <head> — Berisi informasi tentang halaman (judul, pengaturan)
  • <title> — Judul yang muncul di tab browser
  • <body> — Berisi semua isi halaman yang terlihat oleh pengunjung

Bab 3 — Heading dan Paragraf

Heading adalah judul atau sub-judul di halaman web. HTML menyediakan 6 level heading, dari <h1> (paling besar) sampai <h6> (paling kecil).

<h1>Judul Utama (Paling Besar)</h1> <h2>Sub Judul</h2> <h3>Sub-sub Judul</h3> <h4>Judul Level 4</h4>

Paragraf ditulis menggunakan tag <p>. Setiap paragraf akan otomatis punya jarak di atas dan bawahnya.

<p>Ini adalah paragraf pertama. Aku sedang belajar HTML!</p> <p>Ini paragraf kedua. HTML itu menyenangkan!</p>
Tips: Gunakan <h1> hanya sekali di setiap halaman untuk judul utama. Untuk sub-judul, gunakan <h2>, <h3>, dst.

Bab 4 — Teks Tebal, Miring, dan Garis Bawah

Kamu bisa membuat teks lebih menarik dengan format berikut:

<b>Teks ini tebal</b> <strong>Teks ini juga tebal (lebih penting)</strong> <i>Teks ini miring</i> <em>Teks ini juga miring (ditekankan)</em> <u>Teks ini bergaris bawah</u> <br> <!-- Pindah baris baru --> <hr> <!-- Garis horizontal pemisah -->

Contoh penggunaan:

<p>Namaku <strong>Budi</strong> dan aku suka <em>bermain game</em>.</p>

Bab 5 — Membuat Link (Tautan)

Link membuat pengunjung bisa berpindah dari satu halaman ke halaman lain. Gunakan tag <a> dengan atribut href:

<a href="https://google.com">Kunjungi Google</a> <!-- Buka di tab baru --> <a href="https://youtube.com" target="_blank">Buka YouTube</a>

Penjelasan:

  • href — alamat tujuan link
  • target="_blank" — membuka link di tab baru

Bab 6 — Menampilkan Gambar

Untuk menampilkan gambar, gunakan tag <img>. Tag ini tidak perlu ditutup!

<img src="kucing.jpg" alt="Foto kucing lucu" width="300">
  • src — lokasi/alamat file gambar
  • alt — teks pengganti jika gambar tidak muncul
  • width — lebar gambar dalam piksel
Tips: Selalu isi atribut alt! Ini membantu orang yang tidak bisa melihat gambar untuk tetap memahami isi halaman.

Bab 7 — Membuat Daftar (List)

Ada 2 jenis daftar di HTML:

1. Daftar berurutan (Ordered List) — menggunakan nomor:

<ol> <li>Bangun pagi</li> <li>Mandi</li> <li>Sarapan</li> <li>Berangkat sekolah</li> </ol>

2. Daftar tidak berurutan (Unordered List) — menggunakan titik bulat:

<ul> <li>Apel</li> <li>Jeruk</li> <li>Mangga</li> </ul>

Bab 8 — Membuat Tabel

Tabel digunakan untuk menampilkan data dalam bentuk baris dan kolom:

<table border="1"> <tr> <th>Nama</th> <th>Umur</th> <th>Kelas</th> </tr> <tr> <td>Budi</td> <td>10</td> <td>4 SD</td> </tr> <tr> <td>Ani</td> <td>11</td> <td>5 SD</td> </tr> </table>
  • <table> — membuat tabel
  • <tr> — baris tabel (table row)
  • <th> — judul kolom (table header, otomatis tebal)
  • <td> — isi data (table data)

Bab 9 — Membuat Form (Formulir)

Form digunakan untuk mengumpulkan data dari pengguna, seperti nama, email, atau pesan:

<form> <label>Nama:</label> <input type="text" placeholder="Tulis namamu"> <label>Email:</label> <input type="email" placeholder="email@contoh.com"> <label>Pesan:</label> <textarea></textarea> <button type="submit">Kirim</button> </form>

Jenis-jenis input:

  • type="text" — untuk teks biasa
  • type="email" — untuk alamat email
  • type="password" — untuk kata sandi (tersembunyi)
  • type="number" — hanya angka
  • type="checkbox" — kotak centang
  • type="radio" — pilihan (pilih satu)

Bab 10 — Komentar di HTML

Komentar adalah catatan untuk programmer yang tidak terlihat di halaman web. Berguna untuk menjelaskan kode:

<!-- Ini adalah komentar, tidak akan muncul di website --> <h1>Website Keren</h1> <!-- Bagian di bawah ini untuk menu navigasi --> <nav>...</nav>
