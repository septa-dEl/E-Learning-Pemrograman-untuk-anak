Bab 1 — Apa itu Python?

Python adalah bahasa pemrograman yang sangat mudah dibaca, seperti menulis dalam bahasa Inggris biasa. Python digunakan untuk membuat game, aplikasi AI, website, analisis data, dan masih banyak lagi!

# Program Python pertamamu! print("Halo, Dunia!") print("Aku sedang belajar Python!")
Tahukah kamu? Python dinamakan dari acara komedi Monty Python, bukan dari ular!

Bab 2 — Variabel

Di Python, membuat variabel sangat mudah — cukup tulis nama dan isinya:

# Membuat variabel nama = "Budi" umur = 10 tinggi = 140.5 suka_coding = True # Menampilkan variabel print(nama) # Budi print(umur) # 10 print(f"Nama: {nama}, Umur: {umur}")
Keren! Di Python kamu tidak perlu menulis tipe data — Python akan menebaknya secara otomatis!

Bab 3 — Input dan Output

# Meminta input dari pengguna nama = input("Siapa namamu? ") print("Halo, " + nama + "!") # Input angka umur = int(input("Berapa umurmu? ")) print(f"Kamu berumur {umur} tahun") # Kalkulator sederhana a = int(input("Angka pertama: ")) b = int(input("Angka kedua: ")) print(f"Hasil penjumlahan: {a + b}")

Bab 4 — Tipe Data

# String (teks) nama = "Ani" # Integer (bilangan bulat) umur = 12 # Float (bilangan desimal) tinggi = 145.5 # Boolean (benar/salah) sudah_makan = True # List (daftar) buah = ["apel", "jeruk", "mangga"] # Dictionary (kamus data) siswa = { "nama": "Budi", "umur": 10, "kelas": "4 SD" } # Cek tipe data print(type(nama)) # <class 'str'> print(type(umur)) # <class 'int'>

Bab 5 — Kondisi (IF-ELIF-ELSE)

nilai = 85 if nilai >= 80: print("Nilai A - Luar biasa!") elif nilai >= 60: print("Nilai B - Bagus!") else: print("Perlu belajar lagi") # Perhatikan: Python pakai INDENTASI (spasi) # untuk menandai blok kode, bukan kurung { }
Penting! Di Python, indentasi (spasi di awal baris) itu WAJIB. Biasanya pakai 4 spasi.

Bab 6 — Perulangan

# FOR — pengulangan dengan rentang for i in range(1, 6): print(f"Hitungan ke-{i}") # Output: ke-1, ke-2, ke-3, ke-4, ke-5 # FOR — perulangan list hewan = ["kucing", "anjing", "kelinci"] for h in hewan: print(f"Hewan: {h}") # WHILE angka = 1 while angka <= 3: print(angka) angka += 1

Bab 7 — Fungsi

# Membuat fungsi def sapa(nama): print(f"Halo, {nama}!") sapa("Budi") # Halo, Budi! sapa("Ani") # Halo, Ani! # Fungsi dengan return def luas_persegi(sisi): return sisi * sisi hasil = luas_persegi(5) print(f"Luas: {hasil}") # Luas: 25

Bab 8 — List dan Dictionary

# List (daftar) buah = ["apel", "jeruk", "mangga"] buah.append("anggur") # tambah data print(buah[0]) # apel print(len(buah)) # 4 # Dictionary (kamus) profil = { "nama": "Budi", "umur": 10, "hobi": ["coding", "membaca"] } print(profil["nama"]) # Budi print(profil["hobi"][0]) # coding # Menambah data ke dictionary profil["sekolah"] = "SD Maju"
