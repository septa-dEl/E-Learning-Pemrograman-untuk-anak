Bab 1 — Apa itu PHP?

PHP adalah bahasa pemrograman yang berjalan di server (komputer penyedia website). Berbeda dengan JavaScript yang berjalan di browser, PHP bekerja di belakang layar.

PHP bisa digunakan untuk:

  • Membuat sistem login dan registrasi
  • Menyimpan data ke database
  • Mengirim email otomatis
  • Membuat toko online
<?php echo "Halo, Dunia!"; ?>
Tahukah kamu? Website besar seperti Facebook dan Wikipedia awalnya dibuat dengan PHP!

Bab 2 — Variabel di PHP

Di PHP, semua variabel dimulai dengan tanda dolar $:

<?php $nama = "Budi"; $umur = 10; $tinggi = 140.5; echo "Nama: " . $nama; // Nama: Budi echo "Umur: " . $umur; // Umur: 10 ?>
Perbedaan dengan JS: Di PHP pakai $ di depan variabel dan titik . untuk menggabungkan teks.

Bab 3 — Tipe Data

<?php $teks = "Halo"; // String $angka = 42; // Integer $desimal = 3.14; // Float $benar = true; // Boolean $daftar = ["a", "b", "c"]; // Array ?>

Bab 4 — Kondisi IF-ELSE

<?php $nilai = 85; if ($nilai >= 80) { echo "Nilai A - Luar Biasa!"; } elseif ($nilai >= 60) { echo "Nilai B - Bagus!"; } else { echo "Perlu belajar lagi."; } ?>

Bab 5 — Perulangan

<?php // FOR loop for ($i = 1; $i <= 5; $i++) { echo "Hitungan: " . $i . "<br>"; } // FOREACH — untuk array $buah = ["Apel", "Jeruk", "Mangga"]; foreach ($buah as $b) { echo "Buah: " . $b . "<br>"; } ?>

Bab 6 — Fungsi

<?php function spiHalo($nama) { echo "Halo, " . $nama . "!"; } sapiHalo("Budi"); // Halo, Budi! sapiHalo("Ani"); // Halo, Ani! // Fungsi dengan return function tambah($a, $b) { return $a + $b; } echo tambah(5, 3); // 8 ?>

Bab 7 — Array

<?php // Array biasa $warna = ["merah", "biru", "hijau"]; echo $warna[0]; // merah // Array asosiatif (dengan kunci) $siswa = [ "nama" => "Budi", "umur" => 10, "kelas" => "4 SD" ]; echo $siswa["nama"]; // Budi ?>

Bab 8 — Menangkap Data dari Form HTML

<!-- file: form.html --> <form action="proses.php" method="POST"> <input type="text" name="nama"> <button type="submit">Kirim</button> </form> // file: proses.php <?php $nama = $_POST["nama"]; echo "Halo, " . $nama . "! Selamat datang."; ?>
