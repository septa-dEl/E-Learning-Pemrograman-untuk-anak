Kelas JavaScript untuk Pemula

Pelajari cara membuat website menjadi interaktif! JavaScript membuat halaman web bisa bergerak dan merespons.

Bab 1 — Apa itu JavaScript?

JavaScript (disingkat JS) adalah bahasa pemrograman yang membuat halaman web menjadi hidup dan interaktif. Kalau HTML adalah tulang dan CSS adalah kulit, maka JavaScript adalah otaknya!

Dengan JavaScript, kamu bisa:

  • Membuat tombol yang bisa diklik
  • Menampilkan pesan pop-up
  • Mengubah warna elemen saat diklik
  • Membuat game sederhana
  • Menghitung nilai matematika
// Cara menampilkan pesan di JavaScript alert("Halo! Selamat datang di JavaScript!"); console.log("Pesan ini muncul di konsol browser");

Bab 2 — Variabel

Variabel adalah tempat untuk menyimpan data. Bayangkan seperti kotak bernama yang bisa kamu isi apa saja:

// Membuat variabel let nama = "Budi"; let umur = 10; let sukaBelajar = true; // Variabel yang tidak berubah (konstan) const namaSekolah = "SD Maju Jaya"; // Menampilkan isi variabel console.log(nama); // Budi console.log(umur); // 10
Ingat: Gunakan let untuk variabel yang bisa berubah, dan const untuk yang tetap. Hindari var (sudah lama).

Bab 3 — Tipe Data

JavaScript punya beberapa jenis data:

// 1. String (teks) — diapit tanda kutip let nama = "Ani"; // 2. Number (angka) — tanpa tanda kutip let umur = 12; let tinggi = 145.5; // 3. Boolean (benar/salah) let sudahMakan = true; let sedangHujan = false; // 4. Array (daftar/kumpulan data) let buah = ["apel", "jeruk", "mangga"]; // 5. Object (data yang punya properti) let siswa = { nama: "Budi", umur: 10, kelas: "4 SD" };

Bab 4 — Operator

// Operator Matematika let a = 10 + 5; // 15 (tambah) let b = 10 - 3; // 7 (kurang) let c = 4 * 3; // 12 (kali) let d = 20 / 4; // 5 (bagi) let e = 10 % 3; // 1 (sisa bagi) // Operator Perbandingan 5 == 5 // true (sama dengan) 5 != 3 // true (tidak sama) 10 > 5 // true (lebih besar) 3 < 7 // true (lebih kecil) // Menggabungkan teks let salam = "Halo, " + "Budi!"; // "Halo, Budi!"

Bab 5 — Kondisi (IF-ELSE)

Kondisi membuat program bisa mengambil keputusan:

let nilai = 85; if (nilai >= 80) { console.log("Selamat, kamu dapat nilai A!"); } else if (nilai >= 60) { console.log("Lumayan, kamu dapat nilai B."); } else { console.log("Ayo belajar lagi ya!"); }
Analogi: IF-ELSE itu seperti persimpangan jalan. Kalau cuaca cerah, pergi ke taman. Kalau hujan, tetap di rumah.

Bab 6 — Perulangan (Loop)

Loop membuat komputer mengulangi perintah berkali-kali:

// FOR — ulangi sebanyak yang ditentukan for (let i = 1; i <= 5; i++) { console.log("Hitungan ke-" + i); } // Output: Hitungan ke-1, ke-2, ke-3, ke-4, ke-5 // WHILE — ulangi selama kondisi benar let angka = 1; while (angka <= 3) { console.log("Angka: " + angka); angka++; }

Bab 7 — Fungsi

Fungsi adalah kumpulan perintah yang bisa dipanggil berkali-kali:

// Membuat fungsi function spiHalo(nama) { console.log("Halo, " + nama + "!"); } // Memanggil fungsi sapiHalo("Budi"); // Halo, Budi! sapiHalo("Ani"); // Halo, Ani! // Fungsi yang mengembalikan nilai function tambah(a, b) { return a + b; } let hasil = tambah(5, 3); // 8

Bab 8 — Array (Daftar Data)

// Membuat array let hewan = ["kucing", "anjing", "kelinci", "hamster"]; // Mengakses data (mulai dari 0) console.log(hewan[0]); // kucing console.log(hewan[2]); // kelinci // Menambah data hewan.push("ikan"); // Menghitung jumlah console.log(hewan.length); // 5 // Perulangan array for (let i = 0; i < hewan.length; i++) { console.log("Hewan: " + hewan[i]); }

Bab 9 — Interaksi dengan HTML (DOM)

JavaScript bisa mengubah isi halaman web secara langsung:

// Mengubah teks document.getElementById("judul").textContent = "Judul Baru!"; // Mengubah warna document.getElementById("kotak").style.backgroundColor = "blue"; // Menyembunyikan elemen document.getElementById("pesan").style.display = "none";

Bab 10 — Event (Merespons Aksi Pengguna)

// Merespons klik tombol let tombol = document.getElementById("btnKlik"); tombol.addEventListener("click", function() { alert("Tombol diklik!"); }); // Event lainnya: // "mouseover" — saat mouse di atas elemen // "keydown" — saat tombol keyboard ditekan // "submit" — saat form dikirim
