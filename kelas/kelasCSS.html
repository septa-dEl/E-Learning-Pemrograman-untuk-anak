Bab 1 — Apa itu CSS?

CSS adalah singkatan dari Cascading Style Sheets. Kalau HTML adalah kerangka tulang website, maka CSS adalah baju, warna, dan riasan yang membuat website terlihat cantik!

Tanpa CSS, website hanya teks hitam di latar putih. Dengan CSS, kamu bisa mengubah warna, ukuran huruf, jarak antar elemen, dan masih banyak lagi.

Analogi: HTML = struktur rumah (dinding, pintu, jendela). CSS = cat rumah, wallpaper, dekorasi.

Bab 2 — Cara Menulis CSS

Ada 3 cara menulis CSS:

1. Inline CSS — langsung di dalam tag HTML:

<p style="color: blue; font-size: 20px;">Teks biru besar</p>

2. Internal CSS — di dalam tag <style> di head:

<style> h1 { color: red; font-size: 32px; } </style>

3. External CSS (paling direkomendasikan) — file CSS terpisah:

/* file: style.css */ h1 { color: blue; }
<!-- di HTML --> <link rel="stylesheet" href="style.css">

Bab 3 — Warna dan Background

CSS bisa mengubah warna teks dan latar belakang:

body { background-color: #f0f0f0; /* abu-abu terang */ color: #333333; /* teks gelap */ } .kotak-biru { background-color: #3b82f6; color: white; padding: 20px; }

Cara menulis warna di CSS:

  • Nama warna: red, blue, green
  • Kode hex: #ff0000 (merah), #3b82f6 (biru)
  • RGB: rgb(255, 0, 0) (merah)

Bab 4 — Font dan Teks

p { font-family: "Arial", sans-serif; font-size: 16px; font-weight: bold; /* tebal */ text-align: center; /* rata tengah */ line-height: 1.6; /* jarak antar baris */ text-decoration: underline; /* garis bawah */ }
Tips: Gunakan font-size minimal 16px agar mudah dibaca. Untuk anak-anak, 18px lebih baik!

Bab 5 — Box Model

Setiap elemen HTML adalah sebuah "kotak" yang terdiri dari 4 lapisan:

  • Content — isi (teks, gambar)
  • Padding — jarak antara isi dan border
  • Border — garis tepi kotak
  • Margin — jarak antara kotak dengan elemen lain
.kotak { width: 200px; padding: 20px; /* jarak dalam */ border: 3px solid black; margin: 10px; /* jarak luar */ }

Bab 6 — Border dan Shadow

.kartu { border: 2px solid #111; border-radius: 12px; /* sudut bulat */ box-shadow: 4px 4px 10px rgba(0,0,0,0.2); } .lingkaran { border-radius: 50%; /* bikin bulat! */ }

Bab 7 — Flexbox (Tata Letak Fleksibel)

Flexbox membantu mengatur posisi elemen secara horizontal atau vertikal dengan mudah:

.container { display: flex; justify-content: center; /* rata tengah horizontal */ align-items: center; /* rata tengah vertikal */ gap: 16px; /* jarak antar item */ }
Tips: Flexbox sangat berguna untuk membuat navbar, kartu, dan layout modern!

Bab 8 — Hover dan Animasi Sederhana

Buat elemen berubah saat mouse diarahkan (hover):

.tombol { background: #3b82f6; color: white; padding: 12px 24px; border-radius: 8px; transition: all 0.3s ease; /* animasi halus */ } .tombol:hover { background: #1d4ed8; transform: scale(1.05); /* sedikit membesar */ }
