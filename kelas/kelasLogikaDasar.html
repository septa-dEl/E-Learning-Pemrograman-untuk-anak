🚀 Kelas Dasar: Cara Berpikir Komputer

Gratis untuk semua! Mari belajar bagaimana robot dan komputer berpikir sebelum kita mulai mengetik kode.

1. Apa itu Logika? 🧠 2. Urutan (Sequence) 🔢 3. Percabangan (Kondisi) 🔀

🧠 Bab 1 — Komputer itu Pintar atau Bodoh?

Sebenarnya, komputer itu sangat cepat, tapi dia tidak tahu apa-apa sampai kita memberitahunya! Kita harus memberikan instruksi yang sangat jelas. Inilah yang dinamakan Logika Pemrograman.

🤖
Cerita Robot Algo:

Bayangkan kamu punya robot pembuat roti lapis. Kalau kamu bilang "Buat roti lapis!", robot tidak akan mengerti. Kamu harus bilang: "1. Ambil roti, 2. Oleskan selai, 3. Tutup dengan roti lain."

Kunci Utama: Coding / Pemrograman adalah membagi sebuah masalah besar menjadi langkah-langkah kecil yang sangat detail.

🔢 Bab 2 — Urutan (Sequence)

Sequence adalah aturan paling penting. Komputer akan membaca dan menjalankan instruksi dari ATAS ke BAWAH, satu per satu.

Jika urutannya terbalik, hasilnya pasti kacau! Coba lihat contoh memasak mie instan di bawah ini:

✅ URUTAN BENAR:
1. Rebus Air Panas

2. Masukkan Mie

3. Tuang Bumbu

4. Angkat & Makan
❌ URUTAN SALAH:
1. Angkat & Makan

2. Tuang Bumbu

3. Masukkan Mie

4. Rebus Air Panas

Kalau urutannya salah, kita malah makan mie yang masih mentah! Di dunia programming, jika urutannya salah, programnya akan Error (Rusak).

🔀 Bab 3 — Kondisi (Percabangan IF-ELSE)

Di dunia nyata, kita sering membuat keputusan. Misalnya: "Jika di luar hujan, pakaikan payung. Jika tidak, pakai kacamata hitam."

Di komputer, kita menggunakan blok IF (Jika) dan ELSE (Selain itu).

☁️
Mendeteksi Cuaca

Kita minta robot mengecek cuaca sebelum bermain keluar.

IF (Cuaca == Hujan)

Main game di dalam rumah

ELSE

Main bola di lapangan

END IF

Bagaimana membacanya?

  • Jika cuaca benar-benar Hujan, maka robot akan menyuruhmu main game di dalam rumah.
  • ELSE bekerja seperti rencana cadangan. Jika tidak hujan (mungkin cerah atau mendung), robot otomatis menyuruhmu main bola di lapangan.
Maju ke Tahap Selanjutnya! Sekarang kamu sudah menguasai konsep Sequence dan Kondisi IF-ELSE. Selanjutnya kamu bisa membuktikan kehebatanmu dengan memainkan Game Tantangan!
